# 🏦 Banking Application (Full Stack)

A complete full-stack banking system built using modern technologies.

---

## 🚀 Features

### 👤 User Features

* Register & Login (BCrypt authentication)
* Deposit & Withdraw money
* Transfer funds between users
* View transaction history
* Dashboard with recent activity

### 👑 Admin Features

* View all users
* View all transactions
* Admin deposit & withdraw
* Delete users

---

## 🛠️ Tech Stack

### Backend

* Java
* Spring Boot
* Spring Data JPA
* MySQL
* BCrypt Authentication

### Frontend

* React.js
* Tailwind CSS
* React Router

---

## 📂 Project Structure

```
banking-app/
├── backend (Spring Boot)
├── frontend (React App)
```

---

## ⚙️ How to Run

### Backend

```
cd backend
mvn spring-boot:run
```

### Frontend

```
cd frontend
npm install
npm start
```
